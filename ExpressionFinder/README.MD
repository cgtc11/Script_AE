<b>■ExpressionFinder■</b></br>
</br>
<img alt="a" src="https://github.com/cgtc11/Script_AE/blob/main/ExpressionFinder/img.png" /><br>
</br>
使用されてるエクスプレッションを探してリスト表示</br>
リストをダブルクリックで、レイヤー表示してその場所へ飛びます</br>
</br>
レイヤー名が固定されれない状態でNull等が変更された場合</br>
エクスプレッションが切れてしまう為、固定させる機能を追加</br>
</br>
「レイヤー名を現在の名称で固定 (ロック対応)」ボタン</br>
・何も選択せずに押すと、全てを対象</br>
・プロジェクトパネルで素材を選択していたらその素材だけを対象</br>
・タイムラインのレイヤーを選択してたら、そのレイヤー素材だけを対象</br>
</br>
